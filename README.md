# ShellQuicker

ShellQuicker 是一个基于 Godot 4 开发的轻量级图形化 CLI 参数管理工具，专为解决复杂命令行工具（如 FFmpeg, yt-dlp, rsync 等）参数碎片化、难以记忆和复用的痛点而设计。

它将冗长的单行命令解构为可视化的“键值对”结构，支持为每个参数独立添加中文字息，通过 UI 交互降低 CLI 操作的心理负担。

## 核心特性
- **命令可视化解构**：支持将单行长命令一键拆分为垂直排列的参数行，各参数支持独立注释，告别混乱的文本笔记。
- **实时预览与双向定位**：底端预览区动态组装完整指令。编辑参数时，预览区实时高亮对应片段。
- **方便的配置文件与文件树**：可以切换配置文件实现不同工作区的切换。
- **TCP 单实例控制**：通过抢占本地 `58123` 端口实现可靠的单实例运行锁，弥补引擎自带方案在跨平台边界下的不稳定性。
- **编码适配**：内置 `chcp 65001` 注入选项，解决 Windows 环境下执行含中文字符路径命令时的乱码问题。


## 快捷键指南

### 目录管理（左侧面板）
- **`Ctrl + N`**：新建文件夹。
- **`Ctrl + A`**：新建命令模板。
- **`Ctrl + D`**：副本克隆（含所有下属参数和注释）。
- **`F2` / 双击**：重命名。
- **`Delete`**：删除。
- **`Enter`**：执行当前命令（焦点不在输入框时触发）。
- **鼠标交互**：支持跨文件夹自由拖拽排布顺序。

### 命令编辑（右侧上方）
- **智能解析**：粘贴长指令后点击解析按钮，自动生成参数序列。
- **`Ctrl + B`**：在当前行下方插入新参数。
- **排序**：通过左侧 `☰` 图标上下拖动以调整命令组装顺序。

### 合并与（右侧下方）
- **合并选区 (`Ctrl + G`)**：对于解析效果不好的长语句可以后续自定义进行合并。在预览区框选连续文本后按下，可将对应的多行参数合并为单行，便于逻辑重组。

## 技术实现
- **单实例机制**：在 `配置管理器.gd` 的初始化周期中，通过抢占 TCP 端口（58123）实现进程互斥。如需开启多个实例，可在 UI 底部勾选【允许多开】以撤消此锁定。
- **配置持久化**：所有全局设置即时保存于 `settings.cfg`。
